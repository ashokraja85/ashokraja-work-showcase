# Ashokraja Work Showcase

Static portfolio website for showcasing live hosted projects to recruiters.

## Files

- `index.html` - homepage
- `resume.html` - full resume page
- `styles.css` - responsive design
- `app.js` - project list, filters, live screenshot cards

## How to preview locally

Double-click `index.html`, or run:

```powershell
cd E:\Development_zone\ashokraja-work-showcase
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## How to deploy

You can host this as a normal static website:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- Hostinger / cPanel public_html

Upload these four files:

```text
index.html
resume.html
styles.css
app.js
```
